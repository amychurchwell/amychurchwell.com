[![Build Status](https://travis-ci.org/amychurchwell/blog.svg?branch=master)](https://travis-ci.org/amychurchwell/blog)

### installation.


```
 gem install bundler
 ```

 ```
 bundler install
 ```

 ### run.

 ```
 bundle exec jekyll serve
 ```
